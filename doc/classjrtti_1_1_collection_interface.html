Detailed Description

template<typename T>
class jrtti::CollectionInterface< T >

Interface template for native Collection types.

You do not need to worry about this if you are using STL containers to implement your collection. If not, your collection has to specialize a jrtti::iterator for your container types and implement CollectionInterface.
