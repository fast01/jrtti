Detailed Description

Provides a mechanism for custom annotations.

Custom annotations provide metadata to elements of jrtti. This class is a container of Anotation instances wich is stored with the jrtti element. You can retrieve the metadata container by calling annotations method of each jrtti element. jrtti provides the standart Annotation NonStreamable to note that a property is not streamable. You can create your own annotations by creating a class derived from Annotation See sample.h for an example of use
